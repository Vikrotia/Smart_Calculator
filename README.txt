Smart Calculator


Калькулятор реализован на графической библиотеке QT.Есть возможность вызвать 2 вида калькулятора(Обычный, кредитный ), также есть возможность построения графика, для этого введите график функции, укажите диапазон отображаемой области определения и область значения и нажмите кнопку график
Чтобы начать пользоваться калькулятором, его необходимо установить с помощью команды make install. Эта команда установит программу в папку build
Чтобы удалить программу, нужно воспользоваться командой uninstall.
